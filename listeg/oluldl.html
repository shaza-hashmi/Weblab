LISTS IN HTML

ORDERED LIST

Ordered lists displays elements in ordered form. Each item is marked with numbers or letters.It is created using ol tag.Within the ol tag we can use li tag to define item.It is represented by

  1. Numbers
  2. Alphabets
  3. Capital Letters
  4. Small Letters

UNORDERED LIST

It displays elements in bulleted format.It is created using ul tag.We can use li tag to define each item.It is represented using

  • Disc
  • Circle
  • Square
  • Bullets

DEFINTION LIST

It contains items and their description.It utilize dt and dd tag dt defines data items.dd defines data description.dl starts with dt.
